3. 验收客服工单
=========================

网页端操作
----------------
* 您需要处理400 客服代表人员发来的工单验收申请

* 由您判断该工单处理结果是否合格，合格则点击[同意]通过验收；不合格则点击[不同意]退回工单，400 客服代表人员将重新处理

* 您选择[同意]之后，需要回访客户，根据客户的满意程度填写回访表单

* 最后，当所有问题已经妥善处理完成，选择[确认结单]，客户工单工作流程结束；当客户反映问题仍然存在，选择[再次派单]，再次退回工单，400 客服代表人员再次重新处理
